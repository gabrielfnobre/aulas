Formatação de Strings

Qual a importância da formatação de Strings?

Muitas vezes vamos precisar de usar modelos diferentes de strings, seja para concatenar palavras, seja para utilizarmos modelos diferentes de strings para que um determinado código funcione.

Sendo assim, quais são os tipos de strings que existem? E como podemos utilizá-las em JavaScript?

Tipos de formatadores de Strings

Podemos fazer formatações em Strings das seguintes maneiras:

  • Usando áspas simples ou duplas - O uso de áspas dentro do JavaScript sempre é empregado para quando nós queremos usar caracteres;
  • Usando a cráse - O uso da cráse é empregado sempre que nós desejamos identificar um conjunto de palavras como caracteres, porém com a vantagem de poder fazer interpolação e não apenas concatenação;

Como podemos usar formatadores de Strings?

Podemos usar formatadores de Strings das seguintes maneiras:

Áspas duplas ou simples

var n1 = window.prompt("Digite seu nome: ");

var n2 = window.prompt("Digite a sua idade: ");

window.alert("Seu nome é " + n1 + " e você tem " + n2 + " anos.");

(Não importa se você usa áspas duplas ou simples, você vai conseguir gerar cadeias de caracteres do tipo string. Porém, cadeias de caracteres que utilizam áspas não podem fazer interpolação, (Interpolação é quando nós conseguimos adicionar outros elementos dentro de uma única cadeia de caracteres sem a necessidade de usar o sinal de concatenação. No exemplo acima vemos que precisavámos de criar várias cadeias de caracteres abrindo e fechando com áspas e concatenando com variáveis somente para gerar uma única frase, essa é a grande desvantagem das áspas.) somente concatenação. Para verificar como isso acontece na prática, por favor entre neste link.)

Cráse

var n1 = window.prompt(`Digite seu nome: `);

var n2 = window.prompt(`Digite a sua idade: `);

window.alert(`Seu nome é ${n1} e você tem ${n2} anos.`);

(Note que com a utilização da cráse nós precisamos somente de uma cadeia de caracteres e para a fazer a interpolação das variáveis nós utilizamos "place-holder" por através do sinal ${}.(Place-Holder ou "Lugar Reservado", é uma propriedade JavaScript que reserva uma variável para um determinado uso ou formatação em conjunto a outros elementos. Veja que utilizando place-holder dentro da cadeia de caracteres que está cráseado, nós não precisamos de usar o sinal de concatenação e nem ficar usando a cráse a cada conjunto de palavras que fechamos antes de juntar as palavras ás variáveis. Essa é a grande vantagem do uso da cráse em strings, a simplicidade. Sem falar que ela permite o uso de outras sinalizações dentro duma cadeia de caracteres, por exemplo sinalizações que o JavaScript interpretaria como comandos, como por exemplo as próprias áspas.) Para verificar como isso acontece na prática, por favor entre neste link.)
