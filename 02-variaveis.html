Variáveis

O que é uma variável?

Podemos entender bem o conceito de variável usando como exemplo um estacionamento automóveis que atende aos mais variados tipos de veículos, Nele podemos estacionar desde motos até caminhões. O computador é da mesma forma, mas a sua diferença é que o computador guarda dados invés de veículos. Mas assim como não podemos guardar um caminhão na vaga de uma moto, o mesmo se dá com os computadores, onde determinadas variáveis, como as "strings" por exemplo, não podem ser encaradas como "numbers" e assim por diante.

Como criamos uma variável?

Para criar variáveis em JavaScript é muito fácil, desde que tenhamos aberto um espaço no nosso código de programação para o "script" basta nós digitarmos o seletor "var" e dar nome a variável, seguindo depois com o seu Tipo Primitivo.

  • var - usamos esse seletor para abrir o espaço para uma variável;
  • " = " - o sinal de " = " é usado chamado no JavaScript de "recebe", pois ele se trata mesmo de uma forma de identificar um valor que uma determinada variável irá receber;
  • Tipo Primitivo - logo após o sinal de " = " é preciso mostrar ao JavaScript qual é o tipo primitivo do sinal que você vai receber, por exeplo: String, Number e etc;

A declaração de uma variável ficaria mais ou menos assim:

var "n1" (nome dado a variável) = String(window.prompt("Digite seu nome: ")); "essa programação JavaScript faz com que uma janela seja aberta pedindo que a pessoa digite seu nome."

Regras de nomeação das variáveis

Para nomearmos variáveis temos de seguir algumas regras importantes, que são:

  • Podemos começar com letras, $ ou _;
  • Não podemos começar com números;
  • É possível usar letra ou números;
  • É possível usar acentos e símbolos;
  • Não podemos usar espaços;
  • Não podemos usar palavras reservadas;
