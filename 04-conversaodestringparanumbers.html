Conversão de Strings para Numbers

Qual a importância da conversão entre Strings e Numbers?

É de total importância, afinal o JavaScript trabalha com funções numéricas, booleanas, strings e de objetos ao mesmo tempo. A identificação correta do valor que nós estamos trabalhando é vital para que o JavaScript entenda como ele deve trabalhar de acordo com determinado valor.

Por exemplo imagine a confusão no seguinte caso:

var n1 = window.prompt("Digite um número: ");

var n2 = window.prompt("Digite outro número: ");

var s = n1 + n2;

window.alert("A soma dos valores é" + s);

(No exemplo acima temos um script de duas janelas que se abrem em uma página pedindo que o usuário digite duas numerações para serem somadas e uma janela de alerta se abriria revelando a soma, porém, como as janelas foram escritas para tipos primitivos strings, o JavaScript não vai somar os números, em vez disso irá concatená-los. Para ver como isso acontece, click neste link.)

Como resolver esse problema? Convertendo os tipo primitivos strings para numbers.

Como Converter Strings para Numbers?

Para converter strings para numbers nós podemos utilizar 3 comandos conversores:

  • Number.parseInt() - Esse é um conversor de numbers apenas do tipo inteiro, (1, 2, 3...) isso significa que ele nunca mostrará números reais, ou seja, de ponto flutuante.
  • Number.parseFloat() - Esse é um conversor de numbers apenas do tipo real, (1.5, 2.5, 3.5...) isso significa que tem a capacidade de gerar números de ponto flutuante.
  • Number() - Esse é um conversor para numbers tanto do tipo inteiro quanto para numbers tipo ponto flutuante, definí-los como ponto flutuante ou não fica a cargo do navegador.

Como fazemos para utilizá-los em um comando real?

Vejamos exemplos com cada um deles:

"Number.parseInt()":

var n1 = Number.parseInt(window.prompt("Digite um número: "));

var n2 = Number.parseInt(window.prompt("Digite outro número: "));

var s = n1 + n2;

window.alert("A soma dos valores é" + s);

(Note como nós utilizamos o Number.parseInt(), colocamos todo o elemento gerador da variável dentro no conversor Number.parseInt(). Para ver o resultado disso, click neste link.)

"Number.parseFloat()":

var n1 = Number.parseFloat(window.prompt("Digite um número real usando "ponto" e não "vírgula": "));

var n2 = Number.parseFloat(window.prompt("Digite outro número real usando "ponto" e não "vírgula": "));

var s = n1 + n2;

window.alert("A soma dos valores é" + s);

(Note como nós utilizamos o Number.parseFloat(), colocamos todo o elemento gerador da variável dentro no conversor Number.parseFloat(). Para ver o resultado disso, click neste link.)

"Number()":

var n1 = Number(window.prompt("Digite qualquer número, real ou inteiro: "));

var n2 = Number(window.prompt("Digite qualquer número, real ou inteiro: "));

var s = n1 + n2;

window.alert("A soma dos valores é" + s);

(Note como nós utilizamos o Number(), colocamos todo o elemento gerador da variável dentro no conversor Number(). Para ver o resultado disso, click neste link.)
