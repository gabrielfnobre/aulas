Conversão de Numbers para Strings

Qual a importância da conversão entre Strings e Numbers?

É de total importância, afinal o JavaScript trabalha com funções numéricas, booleanas, strings e de objetos ao mesmo tempo. A identificação correta do valor que nós estamos trabalhando é vital para que o JavaScript entenda como ele deve trabalhar de acordo com determinado valor.

Por que converter Numbers em Strings?

Imagine você precisando de obter uma numeração ou conjunto de caracteres numéricos que não representam números de verdade, na verdade se referem a identificação de uma pessoa, um documento ou uma senha, imagine a confusão que seria você tentar concatenar esses números todos juntos, você não quer somá-los, você quer simplesmente juntá-los. Bem, fazer isso com tipos primitivos Number não iria dar certo, por isso você precisa de um conversor de Numbers para Strings.

Veja o exemplo errado de um código que na verdade deseja obter strings e não numbers:

var n1 = Number(window.prompt("Por favor digite o número do dia do seu nascimento: "));

var n2 = Number(window.prompt("Por favor digite o número da casa onde você mora: "));

var s = n1 + n2;

window.alert('Geramos uma senha com a combinação dos números do dia do seu nascimento e da casa onde você mora, a senha é ${s}`);

(Esse código não vai conseguir fazer a concatenação dos números para gerar a senha, por que os valores que as variáveis vão recerber estão convertidos para Numbers em vez de strings, por isso invés de concatenar os valores eles serão somados. Para ver um exemplo prático disso entre neste link.)

Tipos de conversores Strings

  • String( ) - Esse é um modelo de conversor string onde nós colocamos todo o valor gerador da variável dentro dos parêntenses da string;
  • .toString( ) - Esse é um modelo de conversor string onde nós colocamos todo o valor gerador da variável antes do conversor ".toString( )" e deixamos o parêntenses depois dele em aberto;

Como Podemos Utilizar os Conversores String?

String( )

Podemos utilizá-lo da seguinte forma:

var n1 = String(window.prompt("Por favor digite o número do dia do seu nascimento: "));

var n2 = String(window.prompt("Por favor digite o número da casa onde você mora: "));

var s = n1 + n2;

window.alert('Geramos uma senha com a combinação dos números do dia do seu nascimento e da casa onde você mora, a senha é ${s}`);

(Note que desta vez, em vez de usar um conversor "Number()", nós utilizamos um conversor "String()" para fazer a conversão direta dos valores que as variáveis vão receber. (Apesar de sabermos que não era necessária essa conversão, afinal o JavaScript já recebe valores como strings como forma padrão, bastava apenas o "window.prompt()".) Para ver como isso funciona na prática é só entrar neste link.)

.toString( )

Podemos utilizá-lo da seguinte forma:

var n1 = Number(window.prompt("Por favor digite o número do dia do seu nascimento: "));

var n2 = Number(window.prompt("Por favor digite o número da casa onde você mora: "));

var s = n1.toString() + n2.toString();

window.alert('Geramos uma senha com a combinação dos números do dia do seu nascimento e da casa onde você mora, a senha é ${s}`);

(Note que desta vez, mesmo usando o conversor "Number()", ainda conseguimos fazer a conversão por utilizar ".toString()" após as variáveis n1 e n2. Para ver como isso funciona na prática é só entrar neste link.)
